A simple project to demonstrate two types of implementation using [inheritance](https://learn.microsoft.com/en-us/dotnet/csharp/fundamentals/tutorials/inheritance) and [interface](https://learn.microsoft.com/en-us/dotnet/csharp/language-reference/keywords/interface).  
  
Note that C# interface are, by default, NOT serialized by [Unity Serialization](https://docs.unity3d.com/Packages/com.unity.serialization@3.1/manual/index.html) (but [Odin Serializer](https://odininspector.com/odin-serializer) do!).
